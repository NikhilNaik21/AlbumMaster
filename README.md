Photo Album Management Website on AWS EC2

Developed and deployed a Photo Album Management web application that allows users to create albums, upload images, and manage photo collections through an intuitive interface. 
The application is built using ASP.NET, C#, and Microsoft SQL Server, and is hosted on a cloud server using Amazon EC2 from Amazon Web Services.
The system enables users to organize photos into albums, upload and view images dynamically,and manage their profile information. The application was deployed and configured on a Windows server with 
Internet Information Services (IIS), allowing public access through a custom domain.
